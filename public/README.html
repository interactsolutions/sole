Fund Flow Dashboard — Help

This dashboard reads an Excel file on the server. When you replace/update the Excel file, the next refresh will reflect the updated figures.

Run locally

cd server
npm install
export REPORT_PASSWORD='YourStrongPasswordHere'
npm start

Open:

  • Summary: /
  • Details: /details.html (password required)

Update Excel

Replace data/Accounts flow.xlsx and refresh the browser.

Security

Summary is aggregated. Transaction rows are served only after password validation via /api/transactions.
